Logi-PS2 (Play!.js)

Play your legally-owned PS2 games in the browser.

Drop a legally-owned PS2 game (ISO, CSO, CHD, ISZ, BIN, ELF) — the emulator runs locally in your browser.

How this embed works:
This is an embedded instance of the Play!.js web app (hosted by Play! project). It runs in-browser — the emulator runs on the client and the game files you load stay in the browser while playing. If you prefer to self-host the emulator code (Play.js + Play.wasm) so everything is served from your domain, scroll down to the Self-host instructions below.
